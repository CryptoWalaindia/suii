PASSPHRASE

Series 2: Wallets, Keys & Self-Custody

Passphrase

Quick Answer

A passphrase is an optional extra word, phrase, or password added to a seed phrase in
some wallets. It can create an additional layer of wallet protection, but losing it can also
make recovery impossible.

Definition

A passphrase is an extra secret used along with a seed phrase to access a wallet. It is
sometimes called the “25th word,” although it does not have to be one word.

Simple Meaning

A passphrase is like adding a second lock to your wallet backup.
Your seed phrase is one lock.
Your passphrase is another lock.
Both may be needed to restore the same wallet.
This can improve security, but it also increases responsibility.

Why It Matters

Passphrases matter because they can protect a wallet even if a seed phrase is
exposed.
But they are advanced.
If you forget the passphrase, the wallet you created with it may not be recoverable. This
is why beginners should not use passphrases casually.
Passphrase vs Seed Phrase
A seed phrase is the main wallet backup.
A passphrase is an optional extra secret.
Seed phrase = main recovery backup
Passphrase = extra protection layer
If you use a passphrase, you must protect both.

Real-World Example

Asha creates a hardware wallet with a seed phrase.
Then she adds a passphrase.
Later, she tries to restore the wallet. She enters the seed phrase but forgets the
passphrase. The wallet she wants does not appear.
This is why passphrases must be handled carefully.

Common Misunderstanding

Some people think a passphrase is the same as a wallet app password.
It is not.
A wallet app password may unlock an app. A passphrase can change which wallet is
created from a seed phrase.
That makes it much more sensitive.

Safety Note

Do not use a passphrase unless you understand it clearly.
If you use one, back it up carefully and privately. Losing it can mean losing access.

FAQs
Is a passphrase required?
No. It is optional in many wallets. Is a passphrase the same as a password? Not exactly. A passphrase can be part of wallet recovery, while a password may only unlock an app. What happens if I forget my passphrase? You may lose access to the wallet created with that passphrase. Is a passphrase good for security? It can improve security when used correctly, but it also increases complexity. Should beginners use a passphrase? Beginners should first understand seed phrases, backups, and wallet recovery before using a passphrase.
